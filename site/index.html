macOS Tahoe on Proxmox VE

A ready-to-install macOS 26 virtual machine, created with a single command.

$ curl -fsSL https://raw.githubusercontent.com/devfrp/macos-tahoe-proxmox/main/install.sh | bash

Run as root on the Proxmox host. Works on any Intel or AMD CPU, with or without AVX2.

What it does

  1. Checks the host and auto-picks a compatible virtual CPU
  2. Downloads the OpenCore boot loader ISO (adds CryptexFixup on non-AVX2 hosts)
  3. Fetches the official macOS Tahoe recovery from Apple's servers
  4. Creates a fully configured VM (q35, OVMF, VirtIO)

Then

The VM starts by itself: erase the VirtIO disk in Disk Utility (APFS) and install macOS Tahoe. Afterwards, one more command (finalize) makes the VM boot on its own. Full guide in the README.

La VM démarre toute seule : effacer le disque VirtIO dans Utilitaire de disque (APFS) et installer macOS Tahoe. Ensuite, une dernière commande (finalize) rend la VM autonome. Guide complet dans le README.
